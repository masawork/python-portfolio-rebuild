Django + ML Portfolio

このサイト自体が
Django の実装サンプルです

Django REST Framework によるAPI設計、テンプレートによるフロントエンド、 scikit-learn による機械学習推論、OpenAPI(Swagger)による自動ドキュメント生成。 インフラエンジニア12年の経験を持つ坂本が、バックエンド開発力を示すために構築したサイトです。

TECH DEMO

このサイトで実装しているもの

⚙

REST API (DRF)

Projects / Blog / Contact の CRUD API。ページネーション、フィルタリング、検索に対応。

📊

OpenAPI ドキュメント

drf-spectacular による Swagger UI / ReDoc の自動生成。API仕様をインタラクティブに確認可能。

🤖

機械学習推論 API

scikit-learn モデルをロードし、POST リクエストに対してリアルタイム推論結果を返すエンドポイント。

🗃

PostgreSQL + Migration

本番は PostgreSQL、開発は SQLite。Django マイグレーションによるスキーマ管理。

🐳

Docker / Render デプロイ

docker-compose によるローカル開発環境と、Render + gunicorn + WhiteNoise による本番デプロイ。

🔌

テンプレート + Markdown

Django テンプレートエンジンによる SSR。ブログ本文は Markdown で記述し HTML に変換して表示。

API ENDPOINTS

公開 API 一覧

Method Endpoint Description
GET /api/projects/ プロジェクト一覧
GET /api/projects/{id}/ プロジェクト詳細
GET /api/blog/ ブログ一覧
POST /api/contact/ お問い合わせ送信
POST /api/ml/predict/ ML推論実行

全APIの詳細仕様は Swagger UI または ReDoc で確認できます。
