顔認証デモ（ローカル環境での実行）

ℹ️ 重要: 顔認証ライブラリ（opencv-python、face-recognition、dlib）は重いため、Dockerイメージには含まれていません。
デモを実行するには、ローカル環境で上記のライブラリをインストールして、ブログ記事内のサンプルコードを実行してください。

ブログ記事の「ローカル環境での実行方法」セクションを参照して、ローカル環境で顔認証を試すことができます。

ローカル環境での実行手順

  1. 必要なライブラリをインストール： 
    # macOS
brew install cmake
pip install opencv-python face-recognition dlib

# Linux
sudo apt-get install cmake libopenblas-dev liblapack-dev
pip install opencv-python face-recognition dlib
  2. ブログ記事内のサンプルコードをコピーして、`face_recognition_demo.py`として保存
  3. ターミナルで実行： 
    python face_recognition_demo.py
  4. カメラURLを入力（例: 0 または http://192.168.1.100:8080/video）

注意: このデモはローカル環境でのみ実行可能です。Dockerコンテナ内では実行できません。
